Detailed Description

Main header file for Game Config Switcher (GCS).

  • Defines the core data structures for game profiles and file pairs, along with function prototypes for configuration management, file operations, and utility helpers.

typedef struct Profile Profile

Represents a specific Game Profile.

  • A profile consists of a Game Title, a Profile Title (e.g., "TV", "Monitor"), and a linked list of FilePairs associated with that profile.
